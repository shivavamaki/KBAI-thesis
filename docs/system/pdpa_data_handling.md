# PDPA and Data Handling Plan

## หลักการ
งานวิจัยใช้ข้อมูลย้อนหลังจากระบบสารสนเทศโรงพยาบาล โดยต้องปกปิดข้อมูลส่วนบุคคลก่อนนำเข้าสู่ระบบวิเคราะห์

## ข้อมูลที่ต้องปกปิด
- ชื่อและนามสกุล
- HN, VN, visit number, order number
- เลขบัตรประชาชน
- เบอร์โทรศัพท์
- ที่อยู่
- ข้อมูลระบุตัวตนทางตรงอื่น ๆ

## ข้อมูลที่ใช้ได้หลัง de-identification
- อายุ
- น้ำหนัก ส่วนสูง BMI BSA
- ICD diagnosis code
- allergy history ในรูปแบบที่ไม่ระบุตัวตน
- รายการยา ขนาดยา วิธีใช้ ความถี่ ระยะเวลา และสถานะรายการยา
- ผลประเมินของเภสัชกรในรูปแบบ label มาตรฐาน

## ข้อกำหนดด้าน log
- ห้ามเก็บ raw identifier ใน prompt log
- ใช้ `case_id` หรือ `order_id_hash`
- เก็บเฉพาะข้อมูลที่จำเป็นต่อ reproducibility และ audit
